## 🌌 Vision du Portfolio
Ce portfolio n'est pas une simple vitrine, c'est une démonstration technologique de **"L'Ingénierie de l'Éclat"**. Conçu pour refléter la précision du génie logiciel alliée à une esthétique premium et immersive.

### ✨ Fonctionnalités Organiques (Interaction)
- **Ambient Light** : Une lueur diffuse qui suit dynamiquement le curseur pour une immersion totale.
- **Grain Overlay** : Une texture de bruit fractal subtile pour un rendu "papier premium" et haut de gamme.
- **Fluid Physics (Arsenal)** : Des badges technologiques en mouvement constant, réagissant magnétiquement à la souris.
- **Staggered Reveal** : Une typographie qui se dévoile lettre par lettre à l'entrée dans le viewport.

## 📂 Structure du Projet
- `src/components/Hero` : Parallaxe et typographie à fort impact.
- `src/components/Experience` : Chronologie interactive 3D avec reflets.
- `src/components/Skills` : Système de badges flottants avec physique de répulsion.
- `src/components/Contact` : Call-to-action animé et intégration mailto optimisée.

## 🚀 Installation Locale

1. **Cloner le projet**
   ```bash
   git clone https://github.com/Shine831/the-best-of-mr-shine.git
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```
